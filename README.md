# TXBrowser

## Overview

**TXBrowser**，一个实现浏览多种样式的cell框架，轻松并快速实现微博、新闻、App设置、菜谱、商城订单等Cell的框架，支持混合使用。

**note:**本框架主要采用block形式进行处理，因此在使用过程中，开发者要注意内存泄漏问题。
